# DEBT ASSASSINATION

Financial Liberation Protocol — a dark cyberpunk debt tracker with Battle Mode.

## Setup

```bash
cd debt-assassination
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or connect the repo in the Vercel dashboard — it auto-detects Vite.

## Features

- Dashboard: snowball debt tracking, segmented progress bars, payment modal
- Battle Mode: each lender is a villain with health bars and attack animations
- localStorage persistence — never lose data on refresh
- PWA-ready: Add to Home Screen on iOS/Android
- Mobile-first at 375px

## Stack

React 18 · Vite 6 · Pure CSS (no UI libraries)
